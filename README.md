# 🚗 AI-Based Autonomous Navigation System

---

## 📌 Overview

This project simulates an AI agent navigating from a start point to a goal while avoiding obstacles using the A* algorithm.

It represents real-world systems like self-driving cars, warehouse robots, and delivery bots.

---

## 🚀 Key Features

* 🤖 Autonomous navigation using A* algorithm
* 🧠 Shortest path planning
* 🧱 Obstacle avoidance
* 🎮 Real-time simulation (Pygame)
* 📊 Graph visualization (Matplotlib)

---

## 🧠 Tech Stack

* Python
* Pygame
* Matplotlib
* NumPy
