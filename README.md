# 🚗 AI-Based Autonomous Navigation System

---

## 📌 Overview

This project is an AI-based autonomous navigation system that simulates how an intelligent agent moves from a start point to a goal while avoiding obstacles.

It represents real-world applications like self-driving cars, warehouse robots, and delivery systems using path planning and simulation.

---

## 🚀 Key Features

* 🤖 Autonomous navigation using A* algorithm
* 🧠 Intelligent shortest path planning
* 🧱 Obstacle avoidance system
* 🎮 Real-time simulation using Pygame
* 📊 Graph visualization using Matplotlib
* 📍 Start, Goal, and Path tracking
* 📈 Axis-based graph (0.0, 2.5, etc.)

---

## 🔄 System Workflow

Start → Grid Environment → Path Planning (A*) → Obstacle Avoidance → Agent Movement → Goal Reached
